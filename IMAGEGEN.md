# Imágenes generadas para nimbo.mx

Todos los assets de esta versión se generaron con `image_gen`, el generador de
imágenes integrado de Codex. Los originales PNG y la referencia personal usada
para perfilar al personaje se conservan fuera del repositorio. Las fotografías y
la textura publicadas se optimizaron a WebP sin alterar su composición. La tarjeta
Open Graph se publica como PNG de 1200 × 630.
